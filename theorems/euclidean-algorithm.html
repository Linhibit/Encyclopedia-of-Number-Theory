欧几里得算法 (Euclidean Algorithm)

基本原理：

对于两个正整数 $a$ 和 $b$（假设 $a > b$）， 它们的最大公约数等于 $b$ 和 $a \pmod{b}$ 的最大公约数：

$$\gcd(a, b) = \gcd(b, a \bmod b)$$

算法步骤：

  1. 计算 $r = a \bmod b$。
  2. 如果 $r = 0$，则 $b$ 即为最大公约数，算法结束。
  3. 如果 $r \neq 0$，则令 $a = b, b = r$，然后返回步骤 1。

注意： 算法假设 $a > b$，但实际上即使 $a < b$， 第一步计算后也会自动交换两个数的位置。

代码示例 (JavaScript)：

// 使用递归实现
function gcd(a, b) {
    return b === 0 ? a : gcd(b, a % b);
}

// 使用循环实现
function gcdIterative(a, b) {
    while (b !== 0) {
        let temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

// 示例使用
console.log(gcd(48, 18));      // 输出: 6
console.log(gcd(1071, 462));   // 输出: 21
console.log(gcd(17, 13));      // 输出: 1

欧几里得算法的时间复杂度为 $O(\log(\min(a, b)))$， 是一种非常高效的计算最大公约数的方法。
