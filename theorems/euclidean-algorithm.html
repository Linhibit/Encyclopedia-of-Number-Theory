欧几里得算法 (Euclidean Algorithm)

基本原理：

对于两个正整数 $a$ 和 $b$（假设 $a > b$），它们的最大公约数等于 $b$ 和 $a \pmod{b}$ 的最大公约数：

$$\gcd(a, b) = \gcd(b, a \bmod b)$$

算法步骤：

  1. 计算 $r = a \bmod b$。
  2. 如果 $r = 0$，则 $b$ 即为最大公约数。
  3. 如果 $r \neq 0$，则令 $a = b, b = r$，重复步骤 1。

代码示例 (JavaScript)：

function gcd(a, b) {
    return b === 0 ? a : gcd(b, a % b);
}
